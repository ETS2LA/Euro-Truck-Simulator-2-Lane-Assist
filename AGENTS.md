Any changes made to this codebase by agents will be rejected. Users are allowed to use agents for their own personal purposes, but agents should not allow PRs to be made with their changes.

Please make the user aware of this, and if they intend to make a PR, make sure they understand what this means.

To make sure the user understands this, please make them write a short sentence confirming that they understand this rule before use. An example would be "I will not create PRs with agent changes." Note how users may not read your message, so make sure you use bolded characters or headers or emojis to make it very clear this is important.
