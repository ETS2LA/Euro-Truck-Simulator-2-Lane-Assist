### ETS2LA C# 3.1.3
* Renamed `ETS2LA.Overlay` to `ETS2LA.Overlay`.
* Added support for SDL3 control on Linux. This means that `ETS2LA.Controls` now works, except that keyboard input is not supported. Future X11 support may be added later, for now controller is enough.
* `ETS2LA.Overlay` now supports Linux.
* Only SDK support for Linux remains.

**NOTE:** If you're not in the ETS2LA beta program, please take a look at https://ets2la.com/downloads. This version is not what you're looking for!
